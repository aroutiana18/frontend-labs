# Frontend Labs

This repository contains a collection of modern front-end web development projects. The focus of these projects is on creating structured, responsive, and maintainable user interfaces using standard web technologies.

## Repository Structure

The repository is organized into independent project directories to maintain clean separation of concerns.

## Project Overview

### 1. Login Page (Authentication Interface)
A modern, dark-themed login interface designed with a focus on layout control and user experience.

* **Layout:** Implements CSS Flexbox for horizontal and vertical centering of the main form element.
* **Persistent Header:** Features a fixed top navigation bar (`position: fixed`) with a smooth linear gradient spanning from charcoal grey to solid black.
* **UI Elements:** Includes glassmorphism effects via `backdrop-filter`, responsive interactive buttons, and custom link hover states.
* **JavaScript Logic:** Handles form submission handling securely by preventing default page reloads (`e.preventDefault()`) and capturing user inputs dynamically.

### 2. Mini Calculator
A functional, browser-based calculator interface designed for execution of standard arithmetic operations.

* **Architecture:** Adheres to strict separation of HTML structure, CSS presentation, and JavaScript behavior.
* **Interface:** Features an optimized button grid layout utilizing explicit element dimensions for consistent rendering.

## Core Technologies

* **HTML5:** Semantic markup, accessibility compliance, and structural validity.
* **CSS3:** Advanced positioning, flexbox layouts, UI transitions, and custom gradients.
* **JavaScript (ES6+):** Document Object Model (DOM) manipulation, event handling, and form validation.

## Local Installation

To clone and explore this repository locally, execute the following commands in your terminal:

1. Clone the repository:

$ git clone [https://github.com/aroutiana18/frontend-labs.git]

2. Navigate into the project root directory:

$ cd frontend-labs

3. Run the projects:

Open the target index.html file (located inside either PageConnexion or MiniCalculatrice) directly in any standard web browser.
